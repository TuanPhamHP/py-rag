
# PY RAG

Yêu cầu hệ thống (Unbutu):

PYTHON: 3.11

OCR - Tesseract

ChromaDB: ^0.6.0
## Run Locally
Cài dependencies

```bash
  pip install -r requirements.txt
```

Giả lập môi trường (Optional)

```bash
  python3 -m venv venv
  source venv/bin/activate
```

Chạy file main

```bash
  python main.py
```
